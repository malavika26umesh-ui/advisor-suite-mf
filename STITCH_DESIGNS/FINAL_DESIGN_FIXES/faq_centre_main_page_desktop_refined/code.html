Are there any exit loads on Liquid Funds?

As per SEBI regulations, liquid funds carry a tiered exit load if redeemed within 7 days of investment. Redemption on Day 1 carries an exit load of 0.0070%, which progressively reduces to 0.0045% by Day 6. From Day 7 onwards, there is no exit load on liquid fund redemptions.
